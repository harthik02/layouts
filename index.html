2.inline vs internal vs external

        these are the elements for priority such as 
        an inline element is written inside a tag itself
        and a internal element is used to implement css in html itself by style tag 
        and a external element is used to implement css externally 
        such as its precedence as follows 
        inline > internal > external

3.box model

        the box model describes how the elements are structured and how their dimensions are calculated 
        such as these majorly discuss on the following fundamentals :
        content,padding,border,margin

4.margin collapse

        Margin collapse occurs when the vertical margins of adjacent block-level elements or a block-level element 
        and its containing block (parent) overlap. Instead of the margins stacking up, they collapse into a single margin
        such are this include major cases like : 
        adjacent sibling elements , parent and first/last child , empty block-level elements
